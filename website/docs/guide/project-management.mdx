# 项目管理

:::warning 注意
请在运行命令前，确保已经在项目目录下。如果使用了虚拟环境，确保已经激活。
:::

## 插件管理

运行以下命令打开插件管理选单

```shell
nb plugin
```


## 驱动器管理

运行以下命令打开驱动器管理选单

```shell
nb driver
```

## 适配器管理

运行以下命令打开适配器管理选单

```shell
nb adapter
```
